# Lexium landing page

## Run locally

```bash
npm run dev
```

Then open `http://localhost:4173`.

## Deploy

This project is a static site. You can deploy it directly to:

- Vercel
- Netlify
- Cloudflare Pages
- GitHub Pages

All of those support attaching a custom domain after publish.

## Custom domain

Once deployed, connect your registrar domain by adding the DNS records your host gives you, then set the domain as the primary domain in the hosting dashboard.
